# About This Package

This is a MarkdownZip (`.mdz`) package.

- Primary content starts at `index.md`.
- If your tool does not support `.mdz` directly, rename the file to `.zip` or unzip it, then open `index.md` in a Markdown viewer/editor if you have one. A plain text editor will work also.
- Project website: https://markdownzip.org
- Format documentation: https://github.com/kylemwhite/markdownzip-spec
